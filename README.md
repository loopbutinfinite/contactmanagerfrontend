## Contact Manager Frontend

## Description: Made a frontend for the contact manager that is meant to resemble a prototype. The frontend connects to the API backend that connects to the SQL database setup through Microsoft Azure. Technologies & Languages used in this project are ASP.Net, Entity FrameWork Core, Next.js, React, TypeScript, and TSX.
